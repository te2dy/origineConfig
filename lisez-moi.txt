= origineConfig, un plugin pour personnaliser le thème Origine pour Dotclear =

Nom de l’extension : origineConfig
Description : Un plugin de personnalisation du thème Origine
Auteur : Teddy
Licence : GPLv3 ou ultérieure
Adresse de la licence : https://www.gnu.org/licenses/gpl-3.0.fr.html

== Note ==
Cette extension n’aurait pu avoir vu le jour sans les ressources mises à disposition librement par la communauté Dotclear.

== Notes de version ==

=== 1.0.2-beta - 24 mai 2022 ===
- Amélioration : syntaxe dans le code.
- Ajout : dimensions dans le code du logo.
- Mise à jour : chaînes de texte et traduction française.

=== 1.0.1-dev - 16 avril 2022 ===
- Ajout : option pour afficher la première image dans la liste des billets.

- Correction : suppression des paramètres inutilisés dans la base de données.

- Correction : nouvelles options non fonctionnelles.
