= origineConfig, un plugin pour personnaliser le thème Origine pour Dotclear =

Nom de l’extension : origineConfig
Description : Un plugin de personnalisation du thème Origine
Auteur : Teddy
Licence : GPLv3 ou ultérieure
Adresse de la licence : https://www.gnu.org/licenses/gpl-3.0.fr.html

== Note ==
Cette extension n’aurait pu avoir vu le jour sans les ressources mises à disposition librement par la communauté Dotclear.

== Notes de version ==

=== 2.1 - X ===
- Correction : affichage de la valeur par défaut des options à cocher.
- Correction : désactivation des transitions pour des raisons d’accessibilité.
- Modification : adoption d’un système de traduction par identifiant.
- Modification : la classe est text renommée en content-text pour éviter la confusion avec les widgets de texte.
- Suppression : use dcCore superflu.

- Modification : prise en charge de la structure en grille.
- Suppression : options obsolètes.
